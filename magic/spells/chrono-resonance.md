# Chrono Resonance

| | |
|---|---|
| **School** | Temporal |
| **Class** | Temporal Bard (Level 3) |

A Temporal Bard ability that allows usage of magical abilities requiring an instrument without one, and strengthens time-based powers.

## Visual Effect

The bard appears to play an invisible harp in the air, with glowing threads of golden-silver energy materializing for each note.
